# 🛒 BigMart AI — Smart Retail Analytics Platform

> Turning raw retail data into intelligent business decisions using Machine Learning.

---

## 🚀 Project Overview

BigMart AI is a full-stack machine learning application designed to simulate a real-world retail analytics system.
It predicts product sales based on multiple factors and provides a modern dashboard for both users and administrators.

Unlike basic ML projects, this system focuses on **end-to-end implementation** — from model training to deployment with a SaaS-style interface.

---

## ✨ What Makes This Project Unique?

* 🧠 Not just a model — a complete working system
* 🔐 Secure login with role-based access (User & Admin)
* 📊 Interactive dashboard with real-time visualizations
* 🗄 Database-backed activity tracking
* 🌐 Fully deployed web application
* 🎨 Modern UI inspired by real SaaS platforms

---

## ⚙️ Core Features

### 👤 User Features

* Register & login securely
* Predict product sales instantly
* View personal prediction history

### 👨‍💼 Admin Features

* Monitor all users
* Track login activity
* Analyze prediction data
* Filter user-specific records

### 🤖 Machine Learning

* Model: XGBoost (XGBRFRegressor)
* Handles real-world retail dataset
* Provides fast and accurate predictions

## 🏗 How It Works

1. User logs into the system
2. Inputs product details
3. Model predicts sales instantly
4. Data is stored in database
5. Admin can monitor all activity
